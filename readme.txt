Countdown Timer Documentation
Project Description
This is a customizable Countdown Timer built with HTML, CSS, and JavaScript. It allows users to set a countdown in seconds, minutes, or hours, and displays the remaining time in a readable format. The timer includes start, stop, and reset functionality with a notification sound when the time is up.

Features
Set time in seconds, minutes, or hours.
Start, stop, and reset the countdown.
Displays time in a human-readable format (hours, minutes, seconds).
Timer turns red when time is almost up (less than 10 seconds remaining).
Plays a notification sound when the countdown reaches zero.
Responsive design: Works on both desktop and mobile devices.
Accessible: Screen reader-friendly with ARIA labels for better accessibility.
